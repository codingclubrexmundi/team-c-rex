# team-c-rex
website tentang coding club rex mundi


Proyek Coding Club adalah website profil komunitas sekolah yang berfokus pada pembelajaran pemrograman dan teknologi melalui tiga divisi utama: Web Design, Game Development, dan Python Programming. Website ini dibuat menggunakan HTML5 dengan framework Bulma CSS sebagai dasar tampilan, Bootstrap untuk responsivitas tambahan, serta ikon dari Font Awesome. Desain dan efek tambahan diatur lewat file CSS kustom pada folder style/. Sumber aset utama berasal dari CDN Bulma (https://cdn.jsdelivr.net/npm/bulma@1.0.2/css/bulma.min.css), gambar divisi dan proyek dari folder lokal, serta elemen visual seperti logo dan foto pembimbing yang digunakan untuk memperkenalkan aktivitas, anggota, dan karya Coding Club.
